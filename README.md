# RamFiller

A tool for the noble goal of filling up your RAM. Now without needing an
entire web browser.

## Usage

To get started simply run the executable with the amount of gigabytes to full
up:

```sh
# Will allocate 2G
./ramfiller 2
```

For more options run:

```sh
./ramfiller -h
```
